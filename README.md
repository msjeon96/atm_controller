# atm_controller

To clone the repository
```bash
git clone git@github.com:msjeon96/atm_controller.git
```

To run an example user workflow using the atm controller:
```bash
python3 ~/workspace/atm_controller/atm_workflow.py
```

To run test cases validating the atm controller's functionalities
```bash
python3 ~/workspace/atm_controller/test_atm.py
```
